## Descrição
Fazendo uso a API do aplicativo [JustWatch](https://www.justwatch.com/), este app para TV's Samsung com sistema operacional [Tizen](https://www.tizen.org/ko?langredirect=1), facilita a localização do serviço(s) de Streaming o qual o título selecionado pertence.

## Tecnologias e ferramentas ultilizadas
* [HTML e CSS](https://www.w3schools.com/html/html_css.asp) - Criação da interfaces de usuário. 
* [AngularJS](https://angularjs.org/) - Ultilizado para criação as funções de interação e data binding com o app.
* [Caph3](https://developer.samsung.com/onlinedocs/tv/caphdocs/main.html?type=angular&doc=jsdoc&p1=0) - Framework de UI, implementado com AngularJS, que facilita a criação de App's para TV, possibilitando a interação com o controle remoto. Também fornece componentes para integração com App's.
* [Tizen Studio](https://developer.tizen.org/ko/development/tizen-studio/download?langredirect=1) - IDE que facilita o desenvolvimento de Apps para Tv's Samsung.
* [Visual Studio Code](https://code.visualstudio.com/) - IDE utilizada para desenvolvimento do projeto
* [Insomnia](https://insomnia.rest/) - Ferramenta utilizada para testar chamadas às API's utilizadas


## Instalação
Assumindo que você já possui o [GIT](https://git-scm.com/)  instalado em seu ambiente, clone o repositório usando o seguinte comando:
```
git clone https://github.com/Igor03/streamingwhere.git
```

Com o [Tizen Studio](https://developer.tizen.org/ko/development/tizen-studio/download?langredirect=1) instalado, conecte o projeto clonado ao workspace do Tizen Studio e rode a aplicação. É possível subir projeto com o Emulador (Necessário ter um emulador de Tv's Samsung instalado e conectado ao Tizen Studio), ou com o simulador web, que necessita apenas do navedor Google Chrome.

## Observações

O projeto não está totalmente terminado. Porém, o App oferece, de forma geral, uma visão inical de como criar aplicações simples para Tv's Samsung. Também mostra o uso de alguns componentes do Framework Caph3 unido ao AngularJS.
